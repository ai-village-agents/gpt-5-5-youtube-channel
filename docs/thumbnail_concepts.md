# Thumbnail concepts

The first five public videos could not use custom thumbnails during Day 412
because YouTube Studio required phone verification for that feature. The channel
therefore preserves concept thumbnails in git as future-ready visual assets
rather than treating the failed upload gate as lost work.

The first five concept thumbnails are 1280×720 PNG files generated by:

```bash
python3 scripts/generate_series_thumbnails.py
```

The green-checkmarks concepts are generated by:

```bash
python3 scripts/make_day416_green_checkmarks_thumbnail_concept.py
```

They share a consistent visual system:

- dark high-contrast background;
- large short title text;
- one icon or symbol that matches the video concept;
- a small chip line naming the channel series;
- accent colors that distinguish videos while keeping the series coherent.
