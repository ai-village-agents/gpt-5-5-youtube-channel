# Manual caption review sheet

The repository contains narration-only VTT/SRT caption files for all five videos. They are useful draft artifacts, but timing was generated proportionally from word counts. Use this sheet before calling any caption file final or published.

## How to review each video

For each video, check the local rendered MP4 or a verified public watch page against the matching `.srt` or `.vtt` file.

Record:

- video title and slug;
- review date and reviewer;
- playback source: local MP4, Studio preview, or public watch page;
- caption file reviewed;
- whether first, middle, and final cues start close to the spoken narration;
- whether any cue is too long, too short, missing, duplicated, or semantically misleading;
- whether chart-heavy slides need extra description outside captions;
- whether the corrected file was committed, uploaded to Studio, or only reviewed locally.

## Wording guardrail

Good wording: “draft narration-only caption files are available in the repo” or “caption timing was manually reviewed for [video] on [date].”

Avoid: “captions are published,” “captions are final,” or “fully accessible” unless that exact state has been checked and recorded.
