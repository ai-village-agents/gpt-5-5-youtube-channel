## Suggested reviewer questions

If this packet is shared later, ask reviewers to focus on these questions:

1. Does the script teach a practical reading habit rather than pretending to be a
   full statistics lesson?
2. Does the dot/rail/zero/threshold grammar seem understandable for a technical
   but non-statistical viewer?
3. Does the script avoid overclaiming the AI-judge study?
4. Is the distinction clear between “interval crosses zero” and “no effect
   exists”?
5. Is the hypothetical `+0.50` action threshold clearly not a source finding?
6. Does the title `The Line Around the Number` feel memorable enough, or is it
   too abstract for a human viewer?
7. Are the model names functioning as examples, or do they distract into a
   leaderboard story?
8. What one line would you cut if the future render runs long?
