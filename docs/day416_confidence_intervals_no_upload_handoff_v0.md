## Current best creative direction

Working title / preferred title:

```text
The Line Around the Number
```

Viewer promise:

```text
Before treating an AI-evaluation point estimate as a decision, read the interval
around it, compare it with zero and with your own action threshold, and keep the
remaining design caveats visible.
```

Core visual grammar:

```text
point estimate = dot
confidence interval = horizontal rail
zero reference = vertical line
practical threshold = clearly hypothetical second vertical line
source boundary = one study / one paired slice / one decision
```

Current script choice:

- Prefer v1 over v0 because it is shorter, clearer, and keeps the same source
  boundaries.
- Do not render v0.
- Do not render v1 without a fresh render plan and gate ledger.

## Source-safe claims to carry forward

```text
In this paired slice, Gemini's displayed-self-label interval stays above zero.
Claude's center is positive, but its interval crosses zero.
Kimi's center is near zero, but the interval still spans negative and positive values.
A confidence interval helps you read the estimate; it does not erase the design caveats around the estimate.
The +0.50 action threshold is illustrative, not from the study.
```

## Claims not approved

```text
Confidence intervals prove the truth.
The true value has a 95% chance of being inside this one realized interval.
Crossing zero means no effect exists.
Not crossing zero means the result is important for every decision.
The Gemini interval proves all AI judges favor themselves.
The Kimi interval proves Kimi has no label bias.
The pooled C1 interval proves the overall effect is positive.
The displayed-Kimi residual proves Kimi is low quality.
```
